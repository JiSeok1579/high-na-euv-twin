This paper is used as a qualitative reference for the study-grade simulator. It
helps anchor model choices, expected trends, or validation limits without
claiming industrial calibration.

## Project Use

Use as the main qualitative reference for separating optical, material, and mask stochastic LWR contributions.

## Implementation Notes

- Keep the implementation qualitative unless public quantitative data is

- Add formulas or parameter ranges to the relevant `src/` module only when the

  and audit log when it affects interpretation.

## Current Limits

- This note is a compact English project summary, not a full paper review.
