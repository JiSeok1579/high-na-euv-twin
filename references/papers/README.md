# Reference Papers Folder Guide

This folder contains compact English notes for the 21 papers used by the
High-NA EUV lithography simulator.

## Folder Structure

```text
references/
├── high_na_euv_reference_paper_list.txt
├── high_na_euv_paper_search_handoff.md
└── papers/
    ├── INDEX.md
    ├── KNOWLEDGE.md
    ├── QUICK_REFERENCE.md
    ├── README.md
    ├── 01_stochastic_effects_LWR.md
    ├── 02_SHARP_actinic_microscope.md
    ├── ...
    └── 21_statistics_EUV_nanopatterns.md
```

## Reading Order

1. Start with `INDEX.md` for the full paper map.
2. Use `QUICK_REFERENCE.md` for phase-by-phase lookup.
3. Read `KNOWLEDGE.md` for the integrated simulator view.
4. Open individual paper notes when implementing or auditing a module.

## Policy

These notes are project summaries. They support qualitative study-grade
implementation and do not replace reading the original papers.
